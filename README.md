= Proyecto Semestral de Analis de Datos

auth: Bruno Guerra - Maximiliano Mateo

Los archivos del proyecto son:

== EDA.py
Archivo de inicios del curso donde se realizó en análisis exloratorio de datos y renderizado de gráficos.
== Análisis de Frecuencioa.py
Gráficos de frecuencia y detalles de normalización en variables numéricas de características de audio.
== Análisis Hit NoHit.py
Machinelearning para determinar la predicción de hits y análisis de variables más influyentes.
== Dashboard.py
Localhost para analisis exploratorio con controles de filtrado y gráficos.
Ejecutar con: python -m streamlit run dashboard.py
